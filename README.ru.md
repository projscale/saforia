## Данные и хранение

- По умолчанию данные лежат в стандартной директории ОС: macOS `~/Library/Application Support/Saforia`, Windows `%APPDATA%/Saforia`, Linux `~/.local/share/Saforia`.
- Для «портативного» режима задайте переменную окружения `SAFORIA_DATA_DIR=/путь/к/папке` перед запуском — все файлы (мастера, postfixes, бэкапы по умолчанию) будут там.
